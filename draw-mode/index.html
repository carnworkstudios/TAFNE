Table Formatter · Draw Mode

Draw Mode — Build Tables From Raw Text

Convert unstructured raw data, terminal output, and messy text into a clean HTML table. Paste text, paint individual fragments into grid cells, and build your table structure visually — no coding required.

Key Features

  • Interactive blank grid canvas for manual data placement
  • Paint Mode — select text fragments and drop them into the active cell
  • Bulk paste with auto-advance cursor (row or column direction)
  • Dynamic grid resizing — add or remove rows and columns at any time
  • Build Table — convert your painted layout into a styled, editable sheet
  • Handles terminal output, ASCII tables, log files, and raw text lists
